## Dev tools used

- Next v14
- Ant Design
- EsLint
- Husky
- Prettier

## Development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## EsLint

```bash
npm run lint
# or
yarn lint
# or
pnpm lint
# or
bun lint
```

## Prettier Format

```bash
npm run format
npm run format:fix
# or
yarn format
yarn format:fix
# or
pnpm format
pnpm format:fix
# or
bun format
bun format:fix
```
